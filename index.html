Starting-block

- Le framework CSS et HTML de base -

Starting-Block est un Framework CSS et HTML pour débuter un projet sur de bonnes bases. Il se veut flexible pour des designs avancés. Cette page démo vous permet de tester le rendu des styles de base afin de les paramétrer selon vos besoins.

Cette page est responsive. Pour constater comment les classes breakTablet et breakMobile sont utilisées, réduisez la taille de la fenêtre, les blocs modifiés se teinteront d'orange au format tablette et de rouge au format mobile.

Je me suis inspiré de frameworks existants tels que Knacss, ROCSSTI, Bootstrap ou Foundation.

Je m'appelle Thierry Philippe, je suis web designer et intégrateur freelance en région parisienne avec l'accent du Sud-Ouest: Mon portfolio.

Système de grilles

La grille se base sur 12 colonnes flexibles avec une goutière paramétrable en pixel. Il est possible d'imbriquer autant de sets de colonnes que l'on souhaite. Le somme des colonnes col-* et des espaces space-* ne doit pas dépasser 12 sur une ligne!

La classe nogut permet d'annuler les gouttières sur une colonne.
