Starting-block

- Le framework CSS et HTML de base -

Starting-Block est un Framework CSS et HTML pour débuter un projet sur de bonnes bases tout en restant flexible pour des designs avancés. Cette page démo vous permet de tester le rendu des styles de base et de les paramétrer selon vos besoins.

Cette démo est responsive. Pour voir comment les classes break-tablet et break-mobile sont utilisées, réduisez la taille de la fenêtre, les blocs modifiés deviendront orange au format tablette et rouge au format mobile.

Je me suis très largement inspiré de frameworks existants tels que Knacss, ROCSSTI, Bootstrap ou Foundation.

Je m'appelle Thierry Philippe, je suis web designer et intégrateur freelance en région parisienne avec l'accent du Sud-Ouest: Mon portfolio.

Système de grilles

La grille se base sur 12 colonnes flexibles avec une goutière paramétrable. Il est possible d'imbriquer des sets de colonnes à volonté. Le somme des colonnes col-* et des espaces space-* ne doit pas dépasser 12 sur une ligne!

La classe nogut permet d'annuler les gouttières sur une colonne pour des besoins de design.
