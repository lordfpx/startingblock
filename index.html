Starting-block

- The LESS micro framework starter -

Starting-Block is a micro LESS/CSS framework to start your new projects. It is very flexible so that you can work on complexe designs. This page is, indeed, responsive and can be used as a Style Guide.

You better use the LESS files instead of compiled CSS. That way you can use Starting-Block classes into your own styles to keep your HTML lighter.

Some other very good frameworks inspired me, such as: Knacss, ROCSSTI, Bootstrap or Foundation.

Browser support: Internet Explorer 7+, Firefox 3.6+, Safari 3+, Google Chrome 16+.

I'm Thierry Philippe, front-end developper and web designer in Paris... with a pleasant south-wester accent: my portfolio.

Layout Back to top

colset
The columns wrapper. It can contain up to 12 columns and an unfinite nested colset.
col-x-x
They are direct colset children. The first "x" define the media-query breakpoint that will render the column (it can be "s", "m", "l" or "xl"). The second define the size (up to 12 of course).
all-inline-block
Using col-x-x with all-inline-block as parent can be useful in forms or to vertical align children.
space-x-x
To move a column to the right, use this class and define the second "x" as the offset and the first has the break point when it will be activated.
listgrid-x-x
List grid is a way to set up quickly regular grids for galleries or columns that will have same width. Set this class on the parent and direct parent will be sized of "x" column width.
nogutter | all-nogutter
Will remove the gutter of a specific column (or all columns).
